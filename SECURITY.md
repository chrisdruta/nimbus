# Security

Nimbus is a personal, invite-only project, but its source and deployment are
publicly reachable. Please report suspected vulnerabilities privately through
GitHub's **Report a vulnerability** flow rather than opening a public issue.

Do not test against the production deployment in a way that accesses another
member's data, consumes SoundCloud playback quota, or creates sustained load.
A minimal proof of concept against your own account is enough.

Only the current `main` branch is supported. Credentials found in the retired
2020 implementation are historical third-party client IDs and are not used by
the current application.
